Caesar Cipher in Kotlin

This code snippet features the Caesar cipher implemented in the Kotlin programming language. The Caesar cipher encrypts and decrypts messages by shifting the letters in the chosen message by a certain numeric value.

It was originally created for an Android as a cryptography translator app and was a private, collaborative project.

The general process for encryption and decryption in this application is as follows:

  1. receive user input
  2. split the input string into an array
  3. loop through the array
  4. convert the character into a numeric representation of that character
  5. match the numeric representation to a converted value
  6. add the converted value into a final string to return

In order to implement the cipher, I needed to figure out what the algorithm for the cipher was and also how to implement it programmatically. Using the encryption algorithm c = (x + n) mod 26 and the decryption algorithm c = (x - n) mod 26, I converted each letter in a message into a number between 10 and 35 (Unicode character representations of the alphabet in Java) and matched that number to its converted letter. The output is then concatenated together to produce a converted message.

Although this implementation is not perfect, this project gave me an insight into creating real-world applications that may need complex algorithms.

Sparkler

Sparkler is a plugin made for the game Minecraft, version 1.13.2, allowing for players to generate custom items based around US Independence Day.

It was built using Java and the Spigot API, an API that allows for gameplay modification of Minecraft. When the plugin is loaded onto a Minecraft Spigot server, it allows players to use the added functionality from the plugin.

This project began as an idea inspired by and on US Independence Day. My idea was to make it easy for players to celebrate the holiday in Minecraft. The base game does not provide a simple and fast method of creating custom items, leading to the creation of this plugin.

The plugin is activated when a player runs the /sp [item] command in the game. The [item] argument refers to three different options, "banner", "armor", and "firework", which all give a specific custom item. The item is then placed into the player's inventory for he or she to use.
